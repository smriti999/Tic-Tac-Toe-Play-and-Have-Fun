# Tic Tac Toe – Play and Have Fun!!

## Project Description
Tic Tac Toe – Play and Have Fun!! is a simple 3×3 grid game where players compete in Player vs Player or Player vs Computer mode. The project focuses on creating a clean, user-friendly interface with smooth navigation between pages.

---

## Pages Overview

### Home Page
- Display game title
- Play Game button
- How to Play button
- Exit button

### Game Mode Page
- Player vs Player option
- Player vs Computer option
- Start button
- Back button

### Game Board Page
- 3×3 grid
- Display current player turn
- Restart button
- Home button

### Result Page
- Show winner (Player X / Player O)
- Show draw message
- Play Again button
- Home button

### How to Play Page
- Explain basic rules:
  - Game is played on a 3×3 grid
  - Players take turns placing X and O
  - First to get 3 in a row wins
  - If all boxes are filled, it is a draw
- Back button
